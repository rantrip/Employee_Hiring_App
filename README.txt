git clone https://github.com/rantrip/Employee_Hiring_App.git
install mongoDB
go to its bin directory e.g.  c:\Program Files\MongoDB\Server\3.0\bin
mongod
mongo
create database in MongoDB
use employeeDemoDB (to show and switch to employeeDemoDB)
db (to check the DB)

install sts
open sts, create new workspace 
in the project explorer, right click, select import, other general, import existing projectand point to the directory with .project file. 
run maven install 
run demoApplication
go to a browser 
http://localhost:8080/employeeDemo.html
